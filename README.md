# Ticket Manager CLI

A simple command-line ticket manager built with Python. Created for practicing Git workflows.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

## Run Tests

```bash
pytest tests/ -v
```

## Features

- View open tickets
- Add new tickets
- Close tickets
- Persistent storage via JSON
